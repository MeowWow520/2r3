## 2r3

### About
2r3, which means "2d lib renders 3d objects". Just use a simple formula and it can render 3D objects by including [SDL3 Library](https://wiki.libsdl.org/SDL3/FrontPage) and using C++ langange

```
x' = x/z
y' = y/z
```

### Using Library

SDL3: release-3.4.2
